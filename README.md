# 🏪 5 Way House (五味屋) 營運與結帳整合系統 

本系統是一個專為五味屋及其相關據點開發的現代化 Web 應用程式。我們將實體店鋪收銀 POS、倉庫物流、工坊登記、志工排班與住宿管理整合成單一 Web 入口。

系統具備**極度精細的權限隔離 (Fine-Grained RBAC)** 以及**據點地理屏障 (Location-Binding)**，內建了與 Google Drive/Sheets API 的自動日結報表雲端備份機制作業。

---

## 🚀 系統安裝與啟動 (快速部署)

我們提供兩種無痛啟動方式，無論您是開發者還是系統管理員，都能在 3 分鐘內啟動：

### 方法 A：使用自動化腳本 (Mac & Windows 本機推薦)
若您已經安裝了 Python 3.12 與 Git，您可以直接在終端機執行我們寫好的自動化工具：
```bash
# 1. 下載專案原始碼
git clone https://github.com/demonmaycry91/5wayhouse_g.git 5_way_house
cd 5_way_house

# 2. 自動建立虛擬環境、下載套件並生成最高安全級別的 .env 金鑰
bash setup.sh
```

### 方法 B：使用 Docker 一鍵啟動 (正式伺服器推薦)
若您的電腦裝有 Docker Desktop，請直接執行：
```bash
# 系統將自動編譯依賴、建立資料庫表單、並植入測試帳號！
docker-compose --env-file .env up -d --build
```

## 🔑 預設登入帳號

只要是首次全新啟動，系統將自動配置好 **6 個對應不同權限的測試帳號**，密碼均為：**`123456`**

| 帳號 (Username) | 綁定角色 (Role) | 權限能見範圍 |
|------------------|----------------|----------------|
| `admin`          | Admin          | **全域最高管理權限**，可新增據點、修改人員職務與跨區看報表。|
| `cashier`        | Cashier        | 只允許進入收銀台 (POS) 操作與日結，無權查看總體報表。|
| `logistician`    | Logistic       | 進入倉庫物流管理模組。|
| `artisan`        | Workshop       | 進入工坊登記登錄模組。|
| `reception`      | Reception      | 進入住宿時段安排模組。|
| `coordinator`    | Coordinator    | 進入志工與活動管理模組。|

啟動完成後，請打開瀏覽器前往 👉 **`http://localhost:5001`** 登入。

## 📖 詳細系統使用教程

無論您是前線收銀員，還是後台對帳的會計人員，請參考以下的基礎系統操作流程：

### 🛒 1. 收銀員日常操作 (Daily POS)
這是一般門市志工與店長每天必須執行的流程：
1. **每日開帳**：登入系統後點擊「收銀員登入」。找到您負責的店鋪，點按「開始本日營業」，輸入今日撲滿裡有的「開店準備金」即可開門。
2. **客人結帳 (POS)**：點按綠色的「進入 POS 收銀台」。點擊左側對應的商品類別（例如：二手物、雜貨），系統會自動計算總額。若有打折可輸入折扣金，最後輸入客人給的現金，點擊「結帳並記錄」，系統會顯示應找零金額。
3. **每日打烊結帳**：營業結束時，點擊黑色的「日結與盤點現金」。將收銀機內的紙鈔與硬幣分類算好並填入表格，系統會自動比對帳差。最後由雙人電子簽名後，點擊「提交報表」即完成當日工作。

### 📊 2. 會計與總部對帳 (Reporting & OCR)
店長或阿姨每天/每月對帳的操作：
1. **報表查詢**：點擊網頁上方的「報表查詢」按鈕。您可以依照日期與店鋪篩選歷史紀錄，所有數據皆可點擊「匯出 CSV」下載至 Excel 處理。
2. **銀行存款收據驗證 (OCR)**：如果把當日的營業額拿去銀行存，可以針對該日的報表上傳「存款收據照片」。系統會運用 AI 圖片辨識技術 (OCR) 自動判讀收據上的數字，並亮綠燈告訴您金額是否與電腦帳面相符！

### ⚙️ 3. 系統後台管理 (Admin)
僅有 `admin` 帳號可見的操作：
1. **人員管理**：點擊右上角的「系統後台管理 > 使用者管理」。您可以在這裡幫新進員工建立帳號，並「打勾」限制他只能看見哪一間店鋪的機密。
2. **新增據點與商品類別**：當五味屋開了新分店，請至「據點管理」新增該店鋪，並進入「商品類別」設定這間店專屬的收銀機按鈕（例如獨家販售的農產品）。
3. **Google 雲端自動備份**：進入「全域系統設定」，點擊「連結 Google 帳號」。完成登入授權後，未來每天各店的打烊日報表，都會被自動轉成 PDF 檔上傳到公家的 Google Drive 中！
